How to Embed

To embed the XGrid Ambassador Map on your website, simply add this script tag:

<script src="https://main.d3d9zn9ueor3c9.amplifyapp.com/embed.js"></script>

The widget will automatically create a container and load the map.

Custom Container (Optional)

You can also create a custom container with specific dimensions:

<div id="xgrid-ambassador-map" style="width: 100%; height: 600px;"></div>
<script src="https://main.d3d9zn9ueor3c9.amplifyapp.com/embed.js"></script>
